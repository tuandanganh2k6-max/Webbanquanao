# 📖 Introduction

## Purpose
Tài liệu này mô tả yêu cầu phần mềm cho hệ thống website bán quần áo, bao gồm các chức năng, actor và hành vi hệ thống.

## Scope
Hệ thống hỗ trợ mua sắm online và quản lý cửa hàng.

## Definitions
- Guest: Người chưa đăng nhập  
- User (Customer): Người đã đăng nhập  
- Manager: Quản lý cửa hàng  
- Admin: Quản trị hệ thống
