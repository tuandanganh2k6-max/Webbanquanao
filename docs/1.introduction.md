﻿# 📖 Giới Thiệu Dự Án

## 1. Tổng Quan Dự Án

**Tên Dự Án:** Webbanquanao - Nền tảng bán quần áo trực tuyến

**Phiên Bản:** 2.0

**Ngày Cập Nhật:** Tháng 4, 2026

**Tác Giả:** Nhóm phát triển

### Mô Tả Dự Án

Webbanquanao là một ứng dụng web thương mại điện tử (e-commerce) đầy đủ chức năng, cho phép khách hàng mua sắm quần áo trực tuyến. Hệ thống bao gồm:

- **Giao diện khách hàng (Frontend):** Xuyên suốt, dễ sử dụng với React
- **Máy chủ ứng dụng (Backend):** RESTful API với Express.js
- **Cơ sở dữ liệu:** MongoDB lưu trữ thông tin sản phẩm, đơn hàng, người dùng
- **Xác thực:** JWT + bcryptjs

## 4. Kiến Trúc Hệ Thống

`
Frontend (React + Vite)
    ↓
REST API (Express)
    ↓
Backend Services
    ↓
MongoDB Database
`

### Ba Tầng Kiến Trúc

**Tầng Trình Bày (Presentation Layer)**
- React components

- Client-side routing
- Cart state management (React Context)

- Express controllers
- JWT authentication
- Role-based access control
- Business rules

**Tầng Dữ Liệu (Data Layer)**
- MongoDB collections
- Mongoose models
- Data validation
- Database queries

## 5. Các Vai Trò Chính

### 1. Guest (Khách)
- Xem sản phẩm
- Tìm kiếm/lọc
- Đăng ký tài khoản
- Đăng nhập

### 2. Customer (Khách hàng)
- Tất cả quyền Guest
- Thêm sản phẩm vào giỏ
- Thanh toán đơn hàng
- Xem lịch sử đơn hàng
- Theo dõi trạng thái

### 3. Manager (Quản lý)
- Tạo/sửa/xóa sản phẩm
- Quản lý tồn kho
- Xem tất cả đơn hàng
- Cập nhật trạng thái đơn hàng
- Xem báo cáo

### 4. Admin (Quản trị)
- Tất cả quyền Manager
- Tạo tài khoản Manager
- Xem/xóa người dùng
- Gán vai trò

| MongoDB | Cơ sở dữ liệu NoSQL |
