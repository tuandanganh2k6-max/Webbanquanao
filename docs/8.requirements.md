## 4. Ràng Buộc Kỹ Thuật

### Stack
- Frontend: React 19.2.4, Tailwind 3.4.19, Vite 8
- Backend: Node.js, Express 5.2.1
- Database: MongoDB 9.4.1, Mongoose 9.4.1
- Auth: JWT, bcryptjs 3.0.3

### Hosting
- Free/Student tier
- Vercel (frontend)
- Railway/Heroku (backend)
- MongoDB Atlas (database)
- Storage: ~5GB max
