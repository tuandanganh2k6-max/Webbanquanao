# Chương 8: Yêu cầu Hệ thống chi tiết (SRS)

## 8.1 Yêu cầu Chức năng
- Hệ thống xác thực bằng JWT và quản lý phiên làm việc thông qua Cookie/LocalStorage.
- Hỗ trợ lưu trữ dữ liệu tập trung qua MongoDB Atlas.
- API xử lý giỏ hàng đồng bộ theo tài khoản người dùng.

## 8.2 Yêu cầu Phi chức năng
- Bảo mật: Băm mật khẩu bằng thuật toán Bcrypt.
- Hiệu năng: Đảm bảo khả năng xử lý dưới 500ms cho các truy vấn đơn giản.
- Khả dụng: Đảm bảo giao diện hiển thị đồng nhất trên các trình duyệt hiện đại.
