# 🧾 User Stories

## US01 - Đăng ký
As a Guest  
I want đăng ký  
So that tôi có thể mua hàng  

## US02 - Xem sản phẩm
As a Guest  
I want xem sản phẩm  
So that tôi lựa chọn  

## US03 - Thêm giỏ hàng
As a User  
I want thêm sản phẩm  
So that mua nhiều sản phẩm  

## US04 - Thanh toán
As a User  
I want thanh toán  
So that hoàn tất đơn  

## US05 - Quản lý sản phẩm
As a Manager  
I want quản lý sản phẩm  
So that vận hành cửa hàng
