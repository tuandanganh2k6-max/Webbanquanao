# Chương 7: Đặc tả Trường hợp Sử dụng chi tiết

## 7.1 Use Case: Mua hàng (Checkout)
- Tác nhân: Khách hàng.
- Luồng chính: Chọn hàng -> Xem giỏ hàng -> Nhập địa chỉ -> Xác nhận đặt hàng -> Nhận ID đơn hàng.

## 7.2 Use Case: Quản lý Sản phẩm (Inventory Management)
- Tác nhân: Admin.
- Luồng chính: Đăng nhập -> Vào Dashboard -> Chọn module Sản phẩm -> Thêm/Sửa thông tin -> Lưu thay đổi.

## 7.3 Use Case: Quản lý Quảng cáo (Ad Service)
- Tác nhân: Admin.
- Luồng chính: Tiếp nhận đối tác -> Lưu thông tin hợp đồng -> Đăng tải hình ảnh truyền thông -> Kích hoạt hiển thị.
