# 🌐 Hướng dẫn Triển khai Hệ thống (Publishing Guide)

Tài liệu này hướng dẫn chi tiết cách đưa dự án Webbanquanao từ môi trường Local lên môi trường Internet (Production).

## 1. Chuẩn bị Cơ sở dữ liệu Cloud (MongoDB Atlas)
1. Truy cập [MongoDB Atlas](https://www.mongodb.com/cloud/atlas) và đăng ký tài khoản.
2. Tạo một **Shared Cluster (FREE)**.
3. Trong mục **Database Access**, tạo một người dùng (Username/Password).
4. Trong mục **Network Access**, chọn "Allow Access from Anywhere" (0.0.0.0/0).
5. Lấy chuỗi kết nối (Connection String) dạng: `mongodb+srv://<user>:<password>@cluster.mongodb.net/webbanquanao`.

## 2. Lưu trữ mã nguồn trên GitHub
1. Tạo một Repository mới trên GitHub.
2. Đẩy mã nguồn lên bằng các lệnh Git:
   - `git init`
   - `git add .`
   - `git commit -m "Deploy version"`
   - `git push origin main`

## 3. Triển khai Backend lên Render.com
1. Tạo tài khoản tại [Render.com](https://render.com) và kết nối với GitHub.
2. Chọn **New -> Web Service** và chọn repo của dự án.
3. Cấu hình:
   - **Runtime:** Node.
   - **Build Command:** `npm install` (Tại thư mục backend).
   - **Start Command:** `node server.js`.
4. Thiết lập **Environment Variables**:
   - `MONGO_URI`: Dán link từ MongoDB Atlas.
   - `JWT_SECRET`: Mã bí mật của bạn.
   - `PORT`: 10000.
5. Sau khi Deploy, sao chép URL mà Render cung cấp.

## 4. Triển khai Frontend lên Vercel.com
1. Trong mã nguồn Frontend, thay đổi tất cả các đường dẫn gọi API từ `http://localhost:5000` thành URL Backend của Render ở bước trên.
2. Truy cập [Vercel.com](https://vercel.com) và kết nối GitHub.
3. Chọn repo và nhấn **Deploy**. Vercel sẽ tự động nhận diện dự án Vite/React.
4. Sau khi hoàn tất, bạn sẽ nhận được tên miền chính thức (Vd: `webbanquanao.vercel.app`).
