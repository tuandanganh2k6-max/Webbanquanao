# Chương 9: Sơ đồ Trường hợp Sử dụng tổng quát

Dưới đây là sơ đồ Mermaid mô tả các tương tác chính:

```mermaid
useCaseDiagram
    actor "Khách hàng" as User
    actor "Quản trị viên" as Admin
    actor "Đối tác" as Partner

    package "Hệ thống Webbanquanao" {
        usecase "Mua hàng trực tuyến" as UC1
        usecase "Tìm kiếm sản phẩm" as UC2
        usecase "Quản lý đơn hàng" as UC3
        usecase "Quản lý kho hàng" as UC4
        usecase "Quản lý Quảng cáo & Hợp đồng" as UC5
        usecase "Báo cáo doanh số" as UC6
    }

    User --> UC1
    User --> UC2
    
    Admin --> UC3
    Admin --> UC4
    Admin --> UC5
    Admin --> UC6

    Partner --> UC5
```
