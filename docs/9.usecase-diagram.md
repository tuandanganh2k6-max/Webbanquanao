## 2. Luồng Tương Tác

### Khách → Khách Hàng

\\\
Khách
  ├─ Duyệt Sản Phẩm
  ├─ Tìm Kiếm / Lọc
  ├─ Xem Chi Tiết
  └─ Đăng Ký Tài Khoản
     └─ Đăng Nhập → Khách Hàng
\\\

### Khách Hàng Mua Sắm

\\\
Khách Hàng
  ├─ Duyệt / Tìm / Lọc
  ├─ Xem Chi Tiết
  ├─ Thêm Vào Giỏ
  ├─ Xem Giỏ
  │  ├─ Cập Nhật Qty
  │  ├─ Xóa Items
  │  └─ Checkout
  ├─ Thanh Toán
  │  ├─ Nhập Địa Chỉ
  │  └─ Tạo Đơn
  ├─ Xem Đơn Hàng
  └─ Theo Dõi Trạng Thái
\\\

### Manager Quản Lý

\\\
Manager
  ├─ Quản Lý Sản Phẩm
  │  ├─ Tạo
  │  ├─ Sửa
  │  ├─ Xóa
  │  └─ Upload Hình
  ├─ Quản Lý Tồn Kho
  ├─ Quản Lý Đơn Hàng
  │  ├─ Xem Tất Cả
  │  ├─ Cập Nhật Status
  │  └─ Gửi Thông Báo
  └─ Xem Báo Cáo
\\\

### Admin Quản Trị

\\\
Admin
  ├─ Quản Lý Người Dùng
  │  ├─ Tạo Manager
  │  ├─ Xem Danh Sách
  │  ├─ Xóa Tài Khoản
  │  └─ Gán Vai Trò
  └─ Tất Cả Quyền Manager
\\\

---

## 3. Luồng Xử Lý

### Đăng Ký

\\\
Khách Nhập → Validate → Hash MK → Tạo User → Email
\\\

### Thanh Toán

\\\
Checkout Form → Validate → Verify Stock → Tạo Order
    ↓
Cập Nhật Stock → Clear Cart → Gửi Email → Confirm
\\\

### Cập Nhật Trạng Thái

\\\
Manager Chọn → Thay Đổi Status → Update DB → Gửi Email → Khách Hàng
\\\

## 5. Trạng Thái Chuyển Đổi

### Trạng Thái Đơn Hàng

\\\
Pending → Processing → Shipped → Delivered
\\\

### Trạng Thái Tài Khoản

\\\
Chưa Đăng Ký → Đã Đăng Ký → Đang Hoạt Động
                    ↓
              Đang Offline
\\\
