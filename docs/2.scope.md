## 3. Ràng Buộc Kỹ Thuật

### Giới Hạn Kiến Trúc

- **Deployment:** Free/Student tier hosting
- **Database:** MongoDB local hoặc Atlas free tier
- **Storage:** Tối đa ~5GB
- **Concurrent Users:** 1000+
- **Response Time:** < 3 giây
- **Page Load:** < 3 giây

### Công Nghệ Cắt Cố Định

- Frontend: React, Tailwind CSS, Vite
- Backend: Node.js, Express
- Database: MongoDB với Mongoose
- Authentication: JWT + bcryptjs
- Hosting: Vercel/Netlify (Frontend), Heroku/Railway (Backend)

### Giới Hạn Thời Gian

## 5. Giả Định

- **Người Dùng:**
  - Có trình duyệt hiện đại (Chrome, Firefox, Safari, Edge)
  - Kết nối Internet ổn định >= 512 Kbps
  - Hiểu biết cơ bản sử dụng website

- **Hệ Thống:**
  - MongoDB chạy ổn định
  - Node.js runtime có sẵn
  - Git version control sử dụng

- **Kinh Doanh:**
  - Manager duy trì chính xác tồn kho
  - Khách hàng cung cấp địa chỉ hợp lệ
  - Hệ thống hoạt động tại Việt Nam
  - Tương lai có thể mở rộng quốc tế
