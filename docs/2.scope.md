# 📌 System Scope

## In-Scope
- Xem sản phẩm
- Tìm kiếm, lọc
- Đăng ký, đăng nhập
- Giỏ hàng, thanh toán
- Quản lý sản phẩm, đơn hàng, doanh thu

## Out-of-Scope
- Ứng dụng mobile
- Hệ thống giao hàng riêng
- AI recommendation

## Assumptions
- Có internet
- Người dùng biết mua hàng online

## Constraints
- Thời gian học kỳ
- Hiệu năng < 3s
